qsizetype qHash(const QQmlSA::SourceLocation &location, qsizetype seed = 0)

Returns the hash value for location, using seed to seed the calculation.

bool operator!=(const QQmlSA::SourceLocation &lhs, const QQmlSA::SourceLocation &rhs)

Returns true if lhs does not equal rhs, and false otherwise. See SourceLocation::operator== for when two source locations are considered equal.

bool operator==(const QQmlSA::SourceLocation &lhs, const QQmlSA::SourceLocation &rhs)

Returns true if lhs equals rhs, and false otherwise. Two SourceLocations are considered equal if they have the same values for their offset, length, line, and column members.
