Qt Quick Examples - RectangularShadow NeumorphicPanel

Demonstrates RectangularShadow usage.

This example demonstrates the RectangularShadow type, which offers an efficient solution for applying rounded rectangle shadows and glows. The API of the RectangularShadow matches to CSS box-shadow element, with some additions.

Running the Example

To run the example from Qt Creator, open the Welcome mode and select the example from Examples. For more information, see Qt Creator: Tutorial: Build and run.
