Qt Quick Vector Image QML Types

This module was introduced in Qt 6.8.

To use the types in this module, import the module with the following line:

 import QtQuick.VectorImage

Qt Quick Vector Image provides support for displaying vector image files in a Qt Quick scene.

It currently supports the SVG file format.

Qt supports multiple options for displaying SVG files. For an overview and comparison of the different ones, see the documentation of the svgtoqml tool.

QML Types

VectorImage

Loads a vector image file and displays it in a Qt Quick scene
