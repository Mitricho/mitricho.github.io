Supported OpenXR Extensions

This page lists all the OpenXR extensions which are implemented in QtQuick3D.Xr.

Note: These extensions might not be supported on all OpenXR-based systems and, therefore, functionality that depends on the extensions will not be available on these systems.

Khronos extension used in QtQuick3D.Xr

  • XR_KHR_D3D11_enable
  • XR_KHR_D3D12_enable
  • XR_KHR_android_create_instance
  • XR_KHR_android_thread_settings
  • XR_KHR_composition_layer_depth
  • XR_KHR_opengl_enable
  • XR_KHR_opengl_es_enable
  • XR_KHR_vulkan_enable
