Qt QML WorkerScript

The Qml WorkerScript module enables the use of threads in a Qt Quick application.

Use WorkerScript to run operations in a new thread. This is useful for running operations in the background so that the main GUI thread is not blocked.

The WorkerScript supports passing messages between the new thread and the parent thread.

To use the types in this module, import the module with the following line:

 import QtQml.WorkerScript
