QDirListing::sentinel Class

class QDirListing::sentinel

This class was introduced in Qt 6.8.

  • sentinel is part of Input/Output and Networking.

Detailed Description

QDirListing returns an object of this type to signal the end of iteration. Dereferencing a QDirListing::const_iterator that is equal to sentinel{} is undefined behavior.

Note: The "classical" STL algorithms don't support iterator/sentinel, so you need to use C++20 std::ranges algorithms for QDirListing, or else a 3rd-party library that provides range-based algorithms in C++17.

See also QDirListing, QDirListing::const_iterator, and QDirListing::DirEntry.
