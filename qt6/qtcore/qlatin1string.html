QLatin1String Class

QLatin1String is the same as QLatin1StringView. More...

Header: #include <QLatin1String>
CMake: find_package(Qt6 REQUIRED COMPONENTS Core)
target_link_libraries(mytarget PRIVATE Qt6::Core)
qmake: QT += core

Detailed Description

QLatin1String is a view to a Latin-1 string. It's the same as QLatin1StringView and is kept for compatibility reasons. It is recommended to use QLatin1StringView instead.

Please see the QLatin1StringView documentation for details.
