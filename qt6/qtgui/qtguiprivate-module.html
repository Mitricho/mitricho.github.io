Detailed Description

Use the following CMake commands in your CMakeLists.txt to access private Qt GUI APIs:

 find_package(Qt6 REQUIRED COMPONENTS Gui)
 target_link_libraries(mytarget PRIVATE Qt6::GuiPrivate)
