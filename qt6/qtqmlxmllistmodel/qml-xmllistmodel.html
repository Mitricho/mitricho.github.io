Qt XmlListModel

Use the XmlListModel type to create a read-only model from XML data. It can be used as a data source for view elements (such as ListView, PathView, or GridView) and other elements that interact with model data (such as Repeater).

Note: This model does not support XPath queries. It supports simple slash-separated paths and, optionally, one attribute for each element.

To use the types in this module, import the module with the following line:

 import QtQml.XmlListModel
